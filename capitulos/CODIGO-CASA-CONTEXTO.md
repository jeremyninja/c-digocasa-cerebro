# Código Casa®

## Qué es

Código Casa es un estudio propietario de investigación de mercado creado por NINJA sobre las familias dominicanas (2024–2026). Combina tres fuentes de datos:

- **Cuantitativo**: Encuesta a 500 personas en Santo Domingo y Santiago
- **Cualitativo**: 9 grupos focales (+70 horas) segmentados por tipología familiar
- **Social Listening**: Monitoreo de conversación digital (septiembre–diciembre 2025)

El estudio revela las tensiones entre el modelo familiar ideal y la realidad vivida del dominicano: lo que dice vs. lo que hace, lo que hereda vs. lo que quiere construir.

---

## Tipologías familiares estudiadas

Los grupos focales se segmentaron por estructura familiar:

- Biparentales con hijos menores
- Biparentales con hijos adultos
- Monoparentales
- Familias mixtas / recompuestas
- Familias sin hijos (con mascota)
- Familias homoparentales
- Familias extendidas

## Estructura de cada reporte (capítulo)

Cada capítulo sigue un framework repetible:

### 1. Índice
Listado de las 10 tensiones del capítulo + secciones adicionales.

### 2. Tensiones (×10)
Cada tensión sigue este flujo de 7 bloques:

```
TENSIÓN [##] — Título provocador que plantea la contradicción

    ├── HALLAZGO / SET DE DATOS
    │   Datos cuantitativos clave (%, cruces, preguntas de encuesta)
    │   + Consumer Voice (verbatim de grupo focal)
    │   + Cruce de datos (pregunta, base, hallazgo, análisis)
    │
    ├── HALLAZGO SOCIAL LISTENING
    │   Conversación digital: menciones, alcance, sentimiento
    │   + 3 verbatims de redes (TikTok, X/Twitter, Instagram)
    │
    ├── MAPEO DE OPORTUNIDADES
    │   Tabla de 3 columnas:
    │   VERDAD INCÓMODA → OPORTUNIDAD → DETONADOR (pregunta "¿Qué pasaría si...?")
    │
    ├── CASO DE ESTUDIO
    │   Referencia internacional de una marca que activó en un territorio similar
    │
    └── KIT DE RESPUESTA DE MARCA
        Framework para construir narrativa:
        SITUACIÓN → OPORTUNIDAD CLAVE → POSTURA/ROL → VERBO → NUEVA NARRATIVA
        Fórmula: [Tu marca] + quiere + [rol] + pero sabe que + [oportunidad] + por eso + [transformación]
```

### 3. Trend Forecast
3 macro fuerzas + micro fuerzas (micro trends) con:
- Definición del fenómeno
- Hashtags asociados
- Triggers (datos duros con fuente)
- Señales (casos reales, contenido viral, innovaciones)
- Contraste con Código Casa (cómo se manifiesta en RD)

### 4. Framework C.O.D.E.
Metodología para convertir data en acción de marca:

- **C** — Contexto: ¿Qué dice la data sobre esta familia?
- **O** — Oportunidad: ¿Dónde está el gap entre lo que existe y lo que necesitan?
- **D** — Dirección: ¿Cómo creo un producto para esta realidad?
- **E** — Evolución: ¿Cómo activo, mido y escalo?

### 5. Territorios de marca (12)
Espacios estratégicos que ningún competidor habita todavía.

### 6. Ejemplos de aplicación del C.O.D.E.
Aplicación del framework a categorías específicas (banca, seguros, retail, etc.)

## Principios del estudio

- **Say-Do Gap**: La distancia entre lo que el dominicano dice y lo que hace (ej. dice equidad, practica machismo)
- **Dominican Proof**: Validación local de tendencias globales con datos propios
- **Matrifocalidad**: Reconocimiento de que la crianza dominicana es colectiva y centrada en figuras femeninas (madre, abuela, tía)
